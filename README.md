# 🎓 Avery's College Planning Dashboard

A comprehensive college planning and application tracking tool built for **Avery Denney (Class of 2031)**, a pre-veterinary student targeting Animal Science programs with a goal of earning a Doctor of Veterinary Medicine (DVM) degree.

---

## What It Does

Tracks every aspect of the college search and application process in one place — from school research and campus visit evaluations to application checklists, essay tracking, deadline timelines, and financial analysis. Built as a Progressive Web App accessible on any device, with real-time sync via Firebase Firestore.

---

## Key Features

| Feature | Description |
|---|---|
| 🏫 School Comparison | 19+ data points across 20+ pre-vet programs |
| ⭐ Visit Evaluations | Weighted scoring system feeding into final rankings |
| ✅ Application Checklists | Verified essay prompts, test policies, honors college requirements |
| ✍️ Essay Tracker | Status tracking across all schools |
| 💰 Financial Analysis | 4-year cost modeling with merit aid estimates |
| 📊 Admission Strategy | Animal Science advantage data by school |
| 🗓️ Deadline Timeline | Chronological view with urgency alerts |
| 🏛️ Greek Life | Sorority overview for all schools |

---

## Built With

- HTML, CSS, Vanilla JavaScript
- Firebase Firestore (real-time sync)
- Progressive Web App (iPhone home screen support)
